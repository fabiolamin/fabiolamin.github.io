Space Battle


Type: Individual project

Game engine: Unity

Space Battle is a simple 2D multiplayer game that players control a spaceship and need to destroy asteroids around the space.

I was responsible for all programming and game design.


Development highlights:

  • My first multiplayer game using Photon framework as a networking tool
  • Real time score
  • Spaceship selection
  • Asteroids spawn
  • Real time countdown for all players in the room
  • Final scoreboard
  • Spaceship’s movement
  • Spaceship’s shooting
