The Last Survivor


Project: Individual

Game engine: Unity

Platform: PC

The Last Survivor is an action game that the player must kill zombies to advance the rounds. The difficulty from game gets harder as rounds advance.

I was responsible for all programming and game design.

Development highlights:

  • Zombie’s behaviors
  • Zombies spawn
  • Player’s shooting
  • Increased game difficulty
  • Supplies spawn (Health and Ammo)
  • Rounds controller
